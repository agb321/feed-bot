DEVELOPMENT OF IOT-BASED ROBOT FOR REAL-TIME CUSTOMER FEEDBACK

Sentiment Analysis Using Large Language Models

Project Overview

This project focuses on designing and implementing an IoT-based robot capable of collecting real-time customer feedback and analyzing sentiment using advanced AI models known as Large Language Models (LLMs). The robot integrates hardware components such as an ESP32 microcontroller, voice and touch input, and cloud services like ThingSpeak for data logging.

With a sleek interface powered by the Nextion display and intelligent back-end processing via OpenAI APIs, the robot interprets user sentiment in natural language input, helping businesses gain instant insights. This solution aims to revolutionize how customer service data is gathered and interpreted, paving the way for smart, adaptive customer experience tools.
